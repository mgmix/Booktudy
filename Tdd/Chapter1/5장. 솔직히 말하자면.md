# 5장. 솔직히 말하자면

``` $5 + 10CHF = $10(환율이 2:1 일 경우) ```

이 테스트는 어떻게 진행해야할까?
바로 작성하기에는 쉽지 않기 때문에, 작은 단계로 쪼개어 진행한다.

Dollar 객체와 비슷하게 동작하는 Franc 객체를 만든 후 단위가 섞인 덧셈 테스트를 작성하고 돌려보면 될 것 같다.

이를 위해 Dollar 테스트 코드를 복사 한 후 Fance 로 수정하여 진행 한다.

테스트 코드 뿐만이 아닌 모델 객체 코드 까지 복사하여 이름만 수정하여 테스트를 통과한다!



- 큰 테스트를 공략하기 어려우니, 진전을 나타낼 수있는 작은 테스트를 만들기
- 중복을 만들고 조금 고쳐서 테스트를 일단 작성
- 기존에 작성한 코드를 복사하고 수정해서 테스트를 통과
